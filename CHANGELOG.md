# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-07

### Added
- Initial release of AI Chatbot Lab
- Multi-model parallel testing capability
- Support for text and image inputs (multimodal)
- AI image generation with multiple providers (Google, Volcengine/Doubao)
- Qiniu Cloud Storage integration for image hosting
- Comprehensive configuration management
- Dark theme UI
- TypeScript support with full type definitions
- Backend proxy server for API security
- Docker support for containerized deployment
- Deployment scripts (deploy.sh, start.bat)
- Comprehensive documentation (README.md, README.zh-CN.md)
- MIT License

### Technical Stack
- React 19
- TypeScript 5.8
- Vite 6.2
- Tailwind CSS 4.1
- Node.js HTTP Server
- Qiniu SDK

### Documentation
- Added complete README with deployment instructions
- Added Chinese documentation (README.zh-CN.md)
- Added code comments and JSDoc documentation
- Added LICENSE file
- Added environment variable template (.env.example)

### Development
- Removed redundant code and old src directory
- Cleaned up test images and temporary files
- Improved .gitignore and .dockerignore
- Added proper code structure and organization
- Enhanced type safety with TypeScript
